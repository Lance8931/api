API Reference -Release(3.27) Map class ---------------------------------------------- 构造器 Map(mapDiv:Element, opts?:MapOptions) 在一个给定DIV的HTML容器中新创建一个地图 方法 fitBounds(bounds:LatLngBounds|LatLngBoundsLiteral) 返回值：none 说明：设置包含了给定边界的视图 getBounds() 返回值：LatLngBounds 说明：返回当前的边界的经纬度.If more than one copy of the world is visible, the bounds range in longitude from -180 to 180 degrees inclusive.如果地图还没有被初始化(例如，mapType仍然为null)，或者地图中心和放大大小没有被设置，那么结果是为 null 或 undefined. getCenter() 返回值：LatLng 说明：返回地图上可见的中心点位置经纬度坐标. 注意这个经纬度坐标对象是不会被覆盖. getClickableIcons() 返回值：boolean 说明：返回该地图标注是否可点击. A map icon represents a point of interest, also known as a POI. 如果返回值为真，那么这个标注在地图上是可以被点击的. getDiv() 返回：Element getHeading() 返回：number 说明：返回罗盘上的偏移角度. 该值是以地理北极方向为基础计算. getMapTypeId() 返回：MapTypeId | string getProjection() 返回：Projection 说明：返回当前的突出部分?. 如果地图没有被初始化(例如：mapType仍然是null), 那么结果为null. 可以监听 projection_change并检查该值以确保它不为null. getStreetView() 返回：StreetViewPanorama 说明：返回一个相对于地图的默认街景边界，包括一个已经嵌入地图的默认街景或者使用了setStreetView()设置的街景. 改变地图的 stretViewControl(街景控制器)将会反射在已经存在的街景边界. getTilt() 返回：number 说明：返回一个当前地图的倾斜角度，是从视图平面到地图平面的角度. 如果为直视的角度，值就为0,45为45°的角度. 45°的图像只会在设置了地图类型为satellite和hybrid的时候有效，包括在一些地点和一些缩放等级里. 注意：这个方法不会返回用setTilt设置的值.详情请见setTilt. getZoom() 返回：number panBy(x:number, y:number) 返回：None 说明：通过给定的像素距离平移地图的中心点. 如果这个距离小于地图的高度和宽度, 那么这个移动过渡的效果将会是一种平滑的动画. 注意这个地理位置系统是从西往东(x 值)和从北到南(y 值)增加的. panTo(latLng:LatLng | LatLngLiteral) 返回：None 说明：通过给定的地理坐标平移地图的中心点. 如果这个改变小于地图的高度和宽度(即不在当前地图中), 那么这个移动过渡的效果将会是一种平滑的动画. panToBounds(latLngBounds:LatLngBounds | LatLngLiteral) 返回：None 说明：
